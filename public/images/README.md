# Technology Logo Images

To complete the "What I Do Best" section, please add the following technology logo images to the `public/images/` directory:

## Required Images:
- `javascript-logo.png` - JavaScript/TypeScript logo
- `react-logo.png` - React logo  
- `nodejs-logo.png` - Node.js logo
- `html-css-logo.png` - HTML/CSS logo
- `database-logo.png` - Database logo (MongoDB, PostgreSQL, etc.)
- `git-logo.png` - Git version control logo

## Where to get logos:
1. **Official websites**: Download from the official technology websites
2. **Simple Icons**: https://simpleicons.org/ - Free SVG icons for popular brands
3. **DevIcons**: https://devicon.dev/ - Developer tool icons
4. **PNG format**: Convert SVG to PNG if needed, or use online converters

## Recommended sizes:
- 64x64 pixels or 128x128 pixels
- PNG format with transparent background
- White/light colored logos work best with the dark theme

## Current fallback:
If images fail to load, the code will show colored placeholder boxes with the technology name.
